## How to Run

1.  Clone or download the repository.
2.  Open the `index.html` file in any modern web browser.
3.  Allow location access when prompted to see restrooms near you (or let it default to the sample location).

## Usage

1.  **Find Nearby**: Click the "Find Nearby" button to center the map on your location.
2.  **Filter**: Use the chips at the top to filter for specific amenities (e.g., "Clean Only").
3.  **View Details**: Click on a marker or a card to see detailed information about a restroom.
